# Student Record Management System

A simple console-based Student Record Management System developed using Core Java. This project demonstrates Object-Oriented Programming (OOP) concepts and basic CRUD (Create, Read, Update, Delete) operations using ArrayList.

## Features

- Add Student Records
- View All Students
- Update Student Details
- Delete Student Records
- Duplicate ID Validation
- Menu-Driven Console Interface

## Technologies Used

- Java
- ArrayList
- Iterator
- Scanner
- Object-Oriented Programming (OOP)

## Project Structure

```
StudentManager.java
│
├── Student Class
│   ├── id
│   ├── name
│   └── marks
│
└── StudentManager Class
    ├── Add Student
    ├── View Students
    ├── Update Student
    ├── Delete Student
    └── Exit
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/student-record-management-system.git
```

2. Navigate to the project directory

```bash
cd student-record-management-system
```

3. Compile the program

```bash
javac StudentManager.java
```

4. Run the program

```bash
java StudentManager
```
